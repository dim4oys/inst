# Поиск идей для локальных бизнесов

## Почему меняем подход

Hacker News хорошо показывает технологические тренды, но плохо показывает боли локального бизнеса. Для локальных идей нужно смотреть ближе к месту продажи: отзывы, мессенджеры, карты, расписания, локальные сообщества и сервисы записи.

## Основная гипотеза

Лучшие локальные AI-идеи выглядят скучно:

- принять заявку;
- ответить в чате;
- записать клиента;
- напомнить;
- взять предоплату;
- перенести запись;
- заполнить пустой слот;
- попросить отзыв;
- квалифицировать лид.

Именно в таких местах нейросети дают быстрый бизнес-эффект.

## Приоритетные ниши

| Ниша | Почему подходит |
| --- | --- |
| Любительский спорт | Регулярные игры, группы, оплата, замены, уровень игроков |
| Салоны и барбершопы | Запись, no-show, повторные визиты, отзывы |
| Тренеры и студии | Расписание, абонементы, группы, переносы |
| Детские кружки | Родители, повторяющиеся вопросы, платежи, документы |
| Рестораны и банкетные залы | Дорогие заявки, пропущенные звонки, квалификация |
| Домашние услуги | Быстрая реакция на заявку, сбор вводных, фото, адрес |
| Автосервисы | Запись, диагностика, напоминания, согласование работ |

## Сигналы, которые ищем

- В отзывах пишут: "не отвечают", "не дозвониться", "забыли про запись".
- В соцсетях запись ведется вручную через комментарии и личные сообщения.
- Есть расписание в картинке, таблице или закрепленном посте.
- У бизнеса много одинаковых вопросов.
- Есть пустые слоты и неявки.
- Клиенту нужно быстро принять решение: записаться, оплатить, прийти.

## Скоринг для локальных идей

| Критерий | Вопрос |
| --- | --- |
| Pain | Бизнес теряет деньги прямо сейчас? |
| Frequency | Эта ситуация повторяется каждый день или неделю? |
| Buyer | Владелец понимает проблему без долгого объяснения? |
| MVP | Можно сделать первую версию за 1-2 недели? |
| Channel | Можно найти первых клиентов вручную в картах/чатах? |
| AI fit | Нейросеть реально нужна или хватит обычной формы? |
| Retention | Сервис нужен постоянно, а не один раз? |
| Expansion | Можно перенести на другие города и ниши? |
